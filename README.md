# 🎯 Desafio 58: O Jogo da Adivinhação v2.0
Neste projeto, o desafio é vencer o "João da Adivinhação"! O computador escolhe um número aleatório e o jogador deve tentar acertar, recebendo dicas se o número correto é maior ou menor que o palpite atual.

# 🛠️ Funcionalidades
- Geração Aleatória: O computador seleciona um número entre 0 e 10 usando a biblioteca random.
- Loop de Tentativas: Utiliza a estrutura de repetição while para manter o jogo rodando até que o usuário acerte o alvo.
- Feedback Dinâmico: A cada erro, o programa informa se o número secreto é maior ou menor, ajudando o jogador a refinar sua estratégia.
- Contador de Palpites: Ao final, o script exibe o total de tentativas necessárias para a vitória, servindo como uma pontuação de desempenho.
